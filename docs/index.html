Voice for AI coding assistants

npm install -g talkback-cli
GitHub Docs

Instant Audio Feedback

Get spoken notifications without switching windows. Perfect for long-running tasks.

Multiple Voices

Choose from US and British accents with distinct voices for multi-session work.

Multi-Provider TTS

ElevenLabs, OpenAI, Azure, AWS Polly, Google Cloud, or local fallback.

MCP Integration

Works with Claude Code and other MCP-compatible AI assistants.

Fast & Non-Blocking

CLI returns in ~50ms while audio plays in background. No workflow interruption.

Smart Caching

Audio caching reduces API calls and speeds up repeated phrases.

# Speak a message
talkback "Build complete, 0 errors"

# Play a sound
talkback --beep success

# Use a specific voice
talkback -v sam "Tests passed"

# Local TTS fallback
talkback --local "No API needed"
